🎯 TÍNH NĂNG:
--------------

### 📥 EXPORT
- Click nút → Download file Excel ngay
- Chứa tất cả: ID, Câu hỏi, Trả lời, Ngày tạo/sửa
- Format đẹp với column width tự động
- Tên file: knowledge-base-[timestamp].xlsx

### 📤 IMPORT
- Click nút → Chọn file Excel
- Hỗ trợ .xlsx và .xls
- Tối đa 5MB
- Validate từng dòng
- Skip duplicate, báo lỗi chi tiết
- Import thành công → Auto reload data

### 📄 TEMPLATE
- Download mẫu Excel chuẩn
- 2 dòng ví dụ
- Hướng dẫn format đúng

================================================================

🚀 CÁCH SỬ DỤNG:
-----------------

### EXPORT:
1. Đăng nhập Admin Panel
2. Click "Export Excel"
3. File tự động download
4. Mở bằng Excel/LibreOffice/Google Sheets

### IMPORT:
1. Chuẩn bị file Excel (hoặc tải template)
2. Điền dữ liệu:
   - Cột "Câu hỏi": Câu hỏi từ SV
   - Cột "Câu trả lời": Trả lời từ admin
3. Lưu file .xlsx
4. Click "Import Excel"
5. Chọn file
6. Confirm → Đợi kết quả

### TEMPLATE:
1. Click "Template"
2. File knowledge-template.xlsx download
3. Mở file, xem format mẫu
4. Copy format để tạo file import

================================================================

📊 FORMAT EXCEL:
----------------

### Cột bắt buộc:
```
| Câu hỏi | Câu trả lời |
```

### Cột tùy chọn (chỉ khi export):
```
| STT | ID | Ngày tạo | Ngày cập nhật |
```

### Tên cột hỗ trợ:
**Câu hỏi:** Câu hỏi / Cau hoi / Question / keyword
**Câu trả lời:** Câu trả lời / Cau tra loi / Answer / answer

🔧 API ENDPOINTS:
-----------------

### GET /api/knowledge/export
**Auth:** Required (JWT)
**Response:** Excel file binary
**Filename:** knowledge-base-[timestamp].xlsx

### POST /api/knowledge/import
**Auth:** Required (JWT)
**Body:** FormData with 'file' key
**Response:**
```json
{
  "success": true,
  "message": "Import thành công 10 câu trả lời",
  "data": {
    "imported": 10,
    "errors": 2,
    "details": ["Dòng 3: ...", "Dòng 5: ..."]
  }
}
```

### GET /api/knowledge/template
**Auth:** Required (JWT)
**Response:** Template Excel file
**Filename:** knowledge-template.xlsx

💡 USE CASES:
-------------

### 1. Khởi tạo Database
```
Scenario: Bắt đầu project mới
1. Tải template
2. Điền 50-100 câu hỏi phổ biến
3. Import một lần
→ Có sẵn knowledge base hoàn chỉnh
```

### 2. Backup Định kỳ
```
Scenario: Sao lưu định kỳ
Hàng tuần:
1. Export dữ liệu
2. Lưu vào Google Drive
3. Tên: backup-2025-12-20.xlsx
→ An toàn, dễ khôi phục
```

### 3. Cập nhật Hàng Loạt
```
Scenario: Sửa nhiều câu trả lời
1. Export dữ liệu
2. Mở Excel, sửa 20 câu trả lời
3. Xóa cũ (hoặc skip duplicate)
4. Import file mới
→ Cập nhật nhanh
```

### 4. Team Collaboration
```
Scenario: Làm việc nhóm
1. Admin export file
2. Gửi team qua email/Slack
3. Team review & bổ sung
4. Admin import lại
→ Cộng tác hiệu quả
```

### 5. Migration từ hệ thống khác
```
Scenario: Chuyển từ hệ thống cũ
1. Export data từ hệ thống cũ
2. Convert sang format chuẩn
3. Import vào system mới
→ Di chuyển dữ liệu dễ dàng
```

---------------------

1. File size: Max 5MB
   → Đủ cho ~10,000 entries

2. No progress bar
   → Import nhỏ thì ok, lớn thì chờ

3. No preview before import
   → User phải tin file đúng

4. No undo import
   → Phải xóa manual hoặc restore backup

5. No bulk delete
   → Phải xóa từng dòng
