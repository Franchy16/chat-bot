🎯 FLOW HOẠT ĐỘNG:
------------------

User nhập câu hỏi
      ↓
Frontend gửi POST /chat
      ↓
Backend nhận request
      ↓
Search trong Knowledge Base (Fuzzy)
      ↓
   Tìm thấy?
   ├─ YES → Trả về từ Database
   └─ NO  → Hỏi Google Gemini AI
              ↓
         Lưu vào Database
              ↓
         Trả về response
      ↓
Frontend hiển thị
      ↓
Lưu vào LocalStorage

🐛 DEBUG:
---------

Nếu có lỗi:
1. Check console log (F12)
2. Check terminal log
3. Kiểm tra .env
4. Kiểm tra kết nối internet
5. Restart server

Lỗi thường gặp:
- CORS → Đã fix với cors package
- API key invalid → Kiểm tra .env
- Port đã dùng → Đổi PORT trong .env
- Module not found → npm install
