🔄 FLOW HOẠT ĐỘNG:
------------------

### Login Flow:
User → /login.html
  ↓
Nhập credentials
  ↓
POST /api/login
  ↓
Server verify → Tạo JWT
  ↓
Return token
  ↓
Save localStorage
  ↓
Redirect /admin.html

### Protected Access:
User → /admin.html
  ↓
Check localStorage token?
  ├─ No → Redirect /login.html
  └─ Yes → POST /api/verify
           ↓
       Token valid?
       ├─ No → Redirect /login.html
       └─ Yes → Load Admin Panel

### API Call:
GET /api/knowledge
  ↓
Headers: Authorization: Bearer <token>
  ↓
Middleware authenticateToken
  ↓
Token valid?
  ├─ No → 401/403 Error
  └─ Yes → Return data

### Logout:
Click Đăng xuất
  ↓
Confirm
  ↓
Clear localStorage
  ↓
Redirect /login.html

1. ĐỔI CREDENTIALS MẶC ĐỊNH

   - Không chia sẻ

3. HTTPS TRONG PRODUCTION
   - HTTP → Token dễ bị intercept
   - HTTPS → Encrypted

   - Lưu an toàn
   - Không commit Git
   - Version control riêng

🐛 TROUBLESHOOTING:
-------------------

"Không có token":
→ Chưa login, đăng nhập lại

"Token không hợp lệ":
→ Token expired hoặc invalid, login lại

"Tên đăng nhập/mật khẩu không đúng":
→ Check .env file

Loop redirect:
→ Clear localStorage, restart server

API 401/403:
→ Token không được gửi hoặc invalid
