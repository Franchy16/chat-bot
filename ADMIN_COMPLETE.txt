🚀 CÁCH SỬ DỤNG:
-----------------

### Truy cập Admin Panel:

**URL:** http://localhost:8080/admin.html

**Hoặc:** Click icon 🛡️ ở góc phải chatbot

### Thêm câu trả lời mới:

1. Click "Thêm mới"
2. Nhập câu hỏi (VD: "Học phí bao nhiêu?")
3. Nhập câu trả lời (có thể dùng emoji, xuống dòng)
4. Click "Lưu lại"

### Chỉnh sửa:

1. Click nút ✏️ ở dòng muốn sửa
2. Cập nhật thông tin
3. Click "Lưu lại"

### Xóa:

1. Click nút 🗑️ ở dòng muốn xóa
2. Xác nhận trong popup
3. Done!

### Tìm kiếm:

Nhập từ khóa vào ô "Tìm kiếm..." → Kết quả hiện ngay

================================================================

🔧 API ENDPOINTS:
-----------------

Base URL: http://localhost:8080/api

1. GET /knowledge
   → Lấy tất cả câu trả lời
   Response: { success, data, total }

2. POST /knowledge
   Body: { keyword, answer }
   → Thêm câu trả lời mới
   Response: { success, message, data }

3. PUT /knowledge/:id
   Body: { keyword, answer }
   → Cập nhật câu trả lời
   Response: { success, message, data }

4. DELETE /knowledge/:id
   → Xóa câu trả lời
   Response: { success, message, data }
